# Simple Web Portfolio

A small single-page portfolio built with **HTML**, **CSS**, and a bit of **JavaScript**.

## Files

- `index.html` – structure and content
- `styles.css` – layout and styling
- `script.js` – smooth scrolling, dark mode toggle, and dynamic year

## How to run

Open `index.html` in your browser (double-click it or drag it into a browser window). No build step or dependencies are required.
