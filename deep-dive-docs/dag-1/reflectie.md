# Reflectie

* Een LLM is een AI Language model die heel veel parameters heeft, deze zijn afgestemd om ervoor te zorgen dat het resultaat zo correct mogelijk is. Door het stukken tekst (prompt) te geven, een reactie daarop aanmaken door te kijken naar wat voor verwacht dat het antwoord op de prompt zou zijn. In kort: het is een soort autocorrect op seroids.
* Een risico van het gebruiken van een LLM is dat het ervoor zorgt dat je minder denkt, voor code of alledaagse zaken zoals het schrijven van e-mails, hoe meer het gebruikt wordt voor simpele zaken, hoe minder je denkt. Wat naar mijn mening slecht is voor mensen.

## wat is het vershil tussen ChatGPT en Lovable

Het voelde alsof ChatGPT meer ondersteuning nodig had dan Lovable, maar ik kon er wel meer prompts mee gebruiken dan Lovable, aangezien al mijn credits op waren na 1 prompt met Lovable. Het was ook iets moeilijker om mijn code vervolgens te downloaden van Lovable, en ik heb over het algemeen geen idee wat er allemaal in gebeurt, dus het is ook moeilijk aan te passen. Ik ven wel blij met mijn resultaat van Lovable, omdat het resultaat dat ik had gekregen met 1 prompt heel goed was.
